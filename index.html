Miniatures YouTube -- Alpha Wolf Vega

Clique sur le script pour voir les 5 variations. Choisis ta preferee pour chaque video.

Chaque video a 5 miniatures (A, B, C, D, E). Le numero du script (001, 002...) correspond au numero dans ton spreadsheet Pipeline.

Choisis 1 miniature par video, puis uploade-la sur YouTube Studio au moment de publier.

Comment utiliser

  1. Tu ouvres ce fichier dans ton navigateur
  2. Tu reperes le numero du script (001, 002, etc.)
  3. Tu regardes les 5 miniatures
  4. Tu choisis celle qui te plait
  5. Tu uploades cette miniature sur YouTube Studio quand tu publies la video
  6. Le fichier image est dans le meme dossier que cette page
